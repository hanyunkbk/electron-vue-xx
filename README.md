# electron-vue-xx
使用electron、vue、element-ui快速构建项目

```bash
# Clone this repository
git clone https://github.com/hanyunkbk/electron-vue-xx
# Go into the repository
cd electron-vue-xx
# Install dependencies
npm install
# Run the app
npm run vue-cli-service serve
npm run electron .
# Build the app
electron-builder build

Building:
  --mac, -m, -o, --macos   Build for macOS 
  --linux, -l              Build for Linux
  --win, -w, --windows     Build for Windows
  --x64                    Build for x64
  --ia32                   Build for ia32
  --armv7l                 Build for armv7l
  --arm64                  Build for arm64
```
